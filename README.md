## Project description
In this project we created 2 very simple fluid simulators. The resion for 2 is that we found 2 different type of approach for this problem and we wanted to try out both of them. And affter this we added a bit more functionality the one that we liked more. 

The first solution was based on D2Q9 Lattice Boltzmann Method and can be found in the [LBM-D2Q9](./LBM-D2Q9/) folder. \
The second project is a Smoothed-Particle Hydrodynamics based solution and we also added the functionalities to this solution. The source code can be found in [SPH](./SPH/) folder.
